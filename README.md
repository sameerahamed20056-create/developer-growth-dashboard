# Developer Growth Dashboard

A Flask-based web application to track daily developer activities and visualize productivity using charts.

## Features

- Add daily activity logs
- View all logs
- Edit existing logs
- Delete logs
- Dashboard with statistics
- Bar Chart visualization
- Pie Chart visualization
- SQLite database integration

## Technologies Used

- Python
- Flask
- Jinja2
- SQLAlchemy
- SQLite
- Bootstrap
- Matplotlib
